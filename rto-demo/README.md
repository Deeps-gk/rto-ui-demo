# RTO Management System - UI Demo

A frontend-only demonstration of an RTO (Regional Transport Office) Management System built for portfolio purposes.

## Features

- **Home Page**: Overview of RTO services
- **Citizen Login**: Mock authentication system
- **Learner's License Application**: Age verification and test scheduling
- **Driving License Application**: LLR validation and slot booking
- **Vehicle Registration**: Complete vehicle details form
- **Status Check**: Application status tracking
- **Admin Dashboard**: Inspector interface with approval/rejection controls

## Tech Stack

- HTML5
- CSS3 (Responsive Design)
- Vanilla JavaScript
- No backend or database required

## Local Setup

1. Download/clone the project files
2. Open `index.html` in any modern web browser
3. Navigate through different sections using the top menu

## Deployment to Netlify

1. Zip the entire `rto-demo` folder
2. Go to [Netlify](https://netlify.com)
3. Drag and drop the zip file to deploy
4. Your demo will be live instantly!
